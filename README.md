# BabyLog

BabyLog is a privacy-first baby daily log PWA. The hosted app serves static code only; baby records, photos, and recordings stay in the browser's local IndexedDB.

## MVP

- Fast one-handed logging for feeding, diapers, sleep, poop, and crying.
- Local-first storage with Dexie and IndexedDB.
- Today timeline and basic trend panels.
- Poop color warning reminders without diagnosis.
- JSON export.
- PWA install support.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
